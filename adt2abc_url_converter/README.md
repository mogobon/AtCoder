# adt2abc_url_converter

このプロジェクトは、PyQt5を使用して、ADTの問題/解説URLをABCのそれに変換するためのお遊びで作ったユーティリティツールです。

## 必要条件

- Python 3.8以上
- PyQt5

## インストール

1. リポジトリをクローン（あるいはスクリプトをコピーしてローカルに保存）します:

2. 必要な依存関係をインストールします:

    ```bash
    pip install PyQt5
    ```

## 使用方法

1. アプリケーションを実行します:

    ```bash
    python adt2abc_url_converter.py
    ```

2. 画面の指示に従ってURLを変換してください。
